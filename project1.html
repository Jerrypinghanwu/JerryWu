Stock system 股票搜尋系統

  • 目的: 想製做一個股票查詢網站，擁有會員系統、API串接當天股票內容
  • API串接，可於首頁右上打上股票代碼進行搜尋該股票訊息
  • 製作會員系統 - 使用SQLite資料庫，可進行註冊、登入，登入後會有個人股客製化

    使用技術
    1. API (twstock,yfinance)
    2. Django
    3. ORM (MySQL、SQLite)

    工具
    1. API(twstock,yfinance)
    2. Django
    3. ORM (MySQL、SQLite)
